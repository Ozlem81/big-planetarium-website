Accessibility Statement

At Big Planetarium, we are committed to making our website accessible to as many users as possible, including people with disabilities.

Perceivable

  • High colour contrast between text and background.
  • Responsive layout for different screen sizes.
  • Clear headings and structured content.

Operable

  • Keyboard-accessible navigation.
  • Menu toggle supports ARIA attributes.
  • All interactive elements use semantic HTML buttons and links.

Understandable

  • Simple, clear language.
  • Consistent navigation across pages.

Robust

  • Semantic HTML5 elements (header, nav, main, footer).
  • ARIA attributes used appropriately.
  • Tested across multiple screen sizes.
