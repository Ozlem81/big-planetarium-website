# Big Planetarium – Web Development Project

## Overview

Big Planetarium is a responsive multi-page website developed as part of the Web Development Assessment. The project demonstrates the application of HTML5, CSS3, and JavaScript to create an accessible, responsive, and user-friendly educational website about the Solar System.

The website allows users to explore information about selected planets and includes a dedicated accessibility statement page.

---

## Project Objectives

The main objectives of this project were:

- To create a structured multi-page website.
- To apply responsive web design principles.
- To implement accessibility best practices.
- To demonstrate semantic HTML usage.
- To organise files using a clear project structure.
- To implement interactive functionality using JavaScript.

---

## Technologies Used

- HTML5 (Semantic structure)
- CSS3 (Responsive layout, Grid, Styling)
- JavaScript (Interactive navigation menu)
- Live Server (Local testing environment)

---

## Features Implemented

### 1. Responsive Design
- Implemented using CSS Grid.
- Mobile navigation with a toggle button.
- Media queries for screen sizes under 768px.
- Layout adapts to desktop and mobile devices.

### 2. Accessibility (WCAG Principles)

The website follows the four core accessibility principles:

**Perceivable**
- High contrast between text and background.
- Structured headings (h1–h3).
- Clear visual hierarchy.

**Operable**
- Keyboard accessible navigation.
- ARIA attributes used in menu toggle.
- Skip-to-content link implemented.

**Understandable**
- Clear, consistent navigation across pages.
- Simple and readable content structure.

**Robust**
- Semantic HTML5 elements (header, nav, main, footer).
- Valid HTML structure.
- Cross-device testing.

## JavaScript Functionality

JavaScript was used to implement a responsive mobile navigation menu. The menu toggle button:

- Controls visibility of navigation links.
- Updates the `aria-expanded` attribute dynamically.
- Improves accessibility and usability on smaller devices.

## Testing
