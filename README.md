# Big Planetarium Website

This project is a responsive multi-page website developed for the Web Development I assessment.

The website demonstrates:
- Semantic HTML structure
- Responsive design using CSS Grid and media queries
- Accessible navigation with ARIA attributes
- Multi-page structure with individual planet pages
- Basic JavaScript functionality for the mobile navigation menu
